# 구현 기능 목록 

### 입력 

* n대의 자동차 입력 갑 받기 
  * 쉼표 기준 
  * 2대 이상 
  * 각 자동차 이름 5자 이하 


* 시도할 횟수 입력 값 받기  
  * 숫자 검증


### 출력

* 최초 사용자 입력 매세지 출력
  * 자동차 입력 문구 : `경주할자동차이름을입력하세요.(이름은쉼표(,)기준으로구분)`
  * 시도할 횟수 문구 : `시도할회수는몇회인가요?`
  * 사용자 입력 오류 : `[ERROR] : {메세지}`


* 경주 실행 
  * 경주 전 문구 : `실행결과`
  * 진행 문구 : 자동차 별 전진 `{자동차이름}:{이동거리 '-' 만큼}`


* 우승자 
  * 최종 우승자 문구 : `최종 우승자 : ` 
  * 여러명인 경우 `,`로 구분하여 출력

	
### 로직 

*  자동차별 전진 및 멈춤 
  * 랜덤 수 4이상 전진 3이하시 멈춤 


* 우승자 추출
  * 시도할 횟수 만큼 이동한 차량 n개 뽑기 


### 에러 

* 사용자 입력 오류 시
  * `[ERROR]` 출력 후 계속 진행 기능 구현  
  * `IllegalArgumentException` & `IllegalStateException` 명확한 유형 처리
